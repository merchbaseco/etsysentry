# EtsySentry Website

Non-functional frontend prototype for EtsySentry dashboard workflows.
Built with Vite + React Router.

## Scope

- Tabbed homepage views:
  - Listings
  - Keywords
  - Shops
  - Logs
- Dense table-first UX with mock data
- No backend wiring yet

## Run

```bash
bun install
bun run dev
```

Dev server runs on `3100` by default (or next available port if occupied).

## Build

```bash
bun run build
```

## Route Map

- `/` - Listings
- `/keywords` - Keywords
- `/shops` - Shops
- `/logs` - Logs
